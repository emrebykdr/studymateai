# StudyMateAI

StudyMateAI, öğrencilerin ders çalışma süreçlerini daha verimli hale getirmek amacıyla geliştirilmiş yapay zeka destekli bir C# masaüstü uygulamasıdır.  
Uygulama, Ollama üzerinden çalışan yerel yapay zeka modelleri ile iletişim kurarak analiz ve içerik üretme işlemlerini gerçekleştirir.

## Özellikler

- Döküman analiz etme (PDF, Word vb.)
- Döküman içeriğine göre özetleme ve yorumlama
- Dökümanlara dayalı otomatik sınav soruları oluşturma
- Video oynatıcı üzerinden yapay zeka destekli analiz ve yorum
- Ollama entegrasyonu ile yerel yapay zeka modelleriyle çalışma
- Kullanıcı tarafından yapay zeka modeli seçebilme
- Sade ve kullanıcı dostu masaüstü arayüzü

## Kullanılan Teknolojiler

- C#
- .NET
- C# Masaüstü Uygulaması
- Ollama (Yerel yapay zeka modeli çalıştırma)
- Yapay zeka API iletişimi
- Dosya okuma ve metin işleme
- Video oynatıcı bileşenleri

## Kurulum

1. Projeyi bilgisayarınıza indirin veya klonlayın.
2. Bilgisayarınıza Ollama uygulamasını kurun.
3. Kullanmak istediğiniz yapay zeka modellerini Ollama üzerinden indirin.
4. Visual Studio ile projeyi açın.
5. Projeyi çalıştırın.

## Kullanım

- Uygulamaya döküman yükleyerek içerik analizi yapabilirsiniz.
- Analiz edilen dökümanlara göre sınav soruları oluşturabilirsiniz.
- Video oynatıcı üzerinden ders videolarını izlerken yapay zeka yorumlarını görüntüleyebilirsiniz.
- Ayarlar bölümünden kullanılacak yapay zeka modelini seçebilirsiniz.
- Tüm yapay zeka işlemleri yerel modeller üzerinden çalışır.

## Amaç

StudyMateAI, öğrencilerin ders materyallerini yapay zeka desteğiyle daha iyi analiz etmesini, sınavlara daha etkili hazırlanmasını ve bireysel öğrenme süreçlerini desteklemeyi amaçlayan bir eğitim uygulamasıdır.
